BoquilaHUB gets a TUI

BoquilaHUB now ships with a terminal user interface for running AI models directly from the command line.

BoquilaHUB now ships with a terminal user interface. You can run inference, manage models, and monitor results without ever leaving your terminal.

The TUI is designed to be lightweight and accessible, especially for headless servers and remote environments where a graphical interface isn't practical.
